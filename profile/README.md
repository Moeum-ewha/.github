## 🍀 How to Install
```npm install```
<br/>   
<br/>

## 🍀 How To Build
### Front-end
```
npm run build
npm run dev
```
<br/>   
<br/>
<br/>


### Back-end
```
npm run build
npm run start
```
<br/>   
<br/>

## 🍀 How to Test
- https://www.moeum.site 에 들어가서 서비스 테스트 및 이용 가능
- ```git clone```을 통해서 직접 실행도 가능하지만, 보안을 위한 cors 처리로 막힐 가능성이 있음
